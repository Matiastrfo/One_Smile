# DentalManager Pro

Sistema integral de gestión para clínicas odontológicas.

## Descripción General

DentalManager Pro es una aplicación de escritorio orientada a la administración integral de clínicas odontológicas.

### Funcionalidades principales

- Gestión de pacientes
- Turnos y agenda
- Odontograma digital
- Gestión de box
- Control de pagos
- Historial de tratamientos

## Stack Tecnológico

- Python
- PyQt5 / CustomTkinter
- SQLite
- ReportLab
- openpyxl

## Arquitectura

- UI Layer
- Application Layer
- Domain Layer
- Persistence Layer
